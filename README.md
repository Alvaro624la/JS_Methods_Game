## How to clone
Run GitBash and put:
```bash
git clone https://github.com/Alvaro624la/JS_Methods_Game.git
```

## Installation
To install and run this project just type and execute
```bash
npm install
```
and then
```bash
npm run build & npm run build-sass
```
